# Changelog

All notable changes to the ICS / GEO Tools research software are recorded here.

## [0.2.0] — Unreleased

### Added

- exact CEI and CEI-4B replay from 36 preserved Experiment 4/4B rows;
- exact archived within-group rank and aggregate-summary replay;
- deterministic 600-task adversarial GoalProgress benchmark;
- legacy, shuffled-goal, efficiency-only, coherence-only, and
  validity-plus-coherence controls;
- preregistered six-point goal-signal reliability sweep;
- explicit preservation of the failed zero-signal robustness condition;
- fresh-checkout verification and canonical result checksum;
- Apache-2.0 software licensing and CC BY 4.0 research-material licensing;
- citation, Zenodo metadata, and release checklist.
- official creator attribution as David Glowalla.

### Evidence boundary

The historical packages contain placeholder Python entry points. Version 0.2.0
reproduces the archived metric/ranking layer and supplies a new clean-room task
generator; it does not reproduce the historical generator.

## [0.1.0] — 2026-07-20

Initial clean-room reconstruction and primary benchmark contract.
